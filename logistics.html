Prerequisites

The course prerequisites include a strong background in probabilities, statistics, and Bayesian networks (such as factorization, d-separation, colliders, and Markov properties). Familiarity with linear algebra and latent variable modeling is useful but not required.

Grading

Your grade in this course will be composed of the following evaluations:

  • 15%: Problem Set (completed individually)
  • 15%: Paper Summary Assignment (completed individually)
  • 15%: Paper Presentations (completed in groups)
  •   5%: Project Proposal (completed in project groups)
  • 15%: Project Presentation (completed in project groups)
  • 35%: Project Final Report (completed in project groups)
References
The followings are useful textbooks for the course. Suggested readings will be posted after each lecture.
  • Judea Pearl (2009) Causality
  • Miguel A. Hernán, James M. Robins (2020) Causal Inference: What If
  • Jonas Peters, Dominik Janzing, and Bernhard Schlkopf (2017) Elements of Causal Inference: Foundations and Learning Algorithms
